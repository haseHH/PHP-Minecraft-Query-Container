To execute a simple ping request to a Minecraft Java Edition Server navigate to ping.php and provide the server address and port (if not default) via the query parameters. Examples:

Server address only: /ping.php?server=mc.hypixel.net
Server address and port: /ping.php?server=mc.hypixel.net&port=25565
Pre 1.7 legacy server: /ping.php?server=172.31.144.1&pre17=true
